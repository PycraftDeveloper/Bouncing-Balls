# Bouncing Balls

Bouncing Balls is a recreation of the 1994 game Puzzle Bobble!

Itch: https://jebbbo.itch.io/bouncing-balls

**_Aim at bubbles of the same color and shoot!_**

## How To Play

In this game the goal is simple, aim your cannon using the mouse and click to fire a ball. The objective is to try and pop all the bubbles in the level before they reach the bottom. Balls that you fire which make a combination of three or more will pop, earning you points!

There are two levels - good luck!

## Installation
